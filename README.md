# IT4663 - Optimization Planning Labs

This repository is designed for optimization and planning labs. It focuses on illustrating and practicing methods such as linear programming, integer programming, constraint programming, branch and bound, greedy algorithms, heuristics, and local search.

## Purpose
- Support learning and hands-on practice of optimization techniques.
- Provide example problems with input files and solution scripts for each topic.
- Serve as a reference for coursework, projects, and assignments related to planning optimization.

## Repository Structure
- `docs/`: class materials, theoretical notes, explanations, and tutorials for optimization concepts.
- `excercise/`: lab exercises organized by method, including problem inputs and solving scripts.
- `project/ShareARide/`: a sample project applying optimization techniques to a ride-sharing scenario.
- `training/`: additional practice scripts and sample datasets for broader experimentation.

This repository is intended mainly for educational and research purposes.
